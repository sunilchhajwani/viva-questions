# EM & CC Viva Questions

This repository contains a comprehensive collection of sequential Viva-style Short Answer Questions (SAQs) for Emergency Medicine and Critical Care postgraduate examination preparation.

## Contents

- **Cardiovascular Series (2200 Questions)**
  - CV_01: STEMI
  - CV_02: NSTEMI
  - CV_03: Heart Failure & Cardiogenic Shock
  - CV_04: Arrhythmias
  - CV_05: Aortic Emergencies
  - CV_06: Pericardial Emergencies
  - CV_07: Hypertensive Emergencies
  - CV_08: Valvular Emergencies
  - CV_09: Pulmonary Embolism & DVT
  - CV_10: Acute Limb Ischemia
  - CV_11: Cardiac Arrest & Post-ROSC Care

- **Bonus Clinical Presentations**
  - Sjögren's Syndrome Masterclass (Interactive HTML)

## Usage
The .docx files are formatted for printing and study. Each question includes an expected answer and examiner notes.
