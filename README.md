# Master Medical Viva & SAQ Repository

This repository is a comprehensive collection of clinical study materials, viva-style Short Answer Questions (SAQs), and narrative cases for Emergency Medicine and Critical Care postgraduate examination preparation.

## Repository Structure

### [Cardiovascular](./Cardiovascular)
- 2200 Sequential SAQs across 11 modules (STEMI, NSTEMI, Arrhythmias, Heart Failure, etc.)
- Sjögren's Syndrome Masterclass (Interactive HTML)

### [Neurological](./Neurological)
- Meningoencephalitis (200 SAQ + 100 Frontiers)
- Narrative cases: Stroke, ICH, Status Epilepticus, etc.

### [Respiratory](./Respiratory)
- Narrative cases: Asthma, COPD, Pneumonia, ARDS, etc.

### [Infectious_Diseases](./Infectious_Diseases)
- Sepsis & Septic Shock (200 SAQ)
- Narrative cases: Meningitis, Necrotizing Fasciitis, etc.

### [Mega-SAQ Sets (1000+ Questions Each)](./)
- [**Orthopaedics**](./Orthopaedics/Orthopaedics_1000_SAQ.docx)
- [**ENT**](./ENT/ENT_1000_SAQ.docx)
- [**Dermatology**](./Dermatology/Dermatology_1000_SAQ.docx)
- [**Ophthalmology**](./Ophthalmology/Ophthalmology_1000_SAQ.docx)
- [**Urology**](./Urology/Urology_1000_SAQ.docx)

### Other Systems
- **Gastrointestinal**: Upper/Lower GI bleed, Pancreatitis, etc.
- **Nephrology**: AKI, Dialysis emergencies, Electrolyte derangements.
- **Trauma**: Multi-trauma, Head/Chest/Abdominal trauma.
- **Toxicology**: Common overdoses, Toxidromes.
- **Endocrine, OBGYN, Psychiatry, Hematology/Oncology.**
- **Environmental Emergencies**: Hypothermia, Heat stroke, Drowning.

## Usage
The .docx files are formatted for printing and study. Each question includes an expected answer and examiner notes.
